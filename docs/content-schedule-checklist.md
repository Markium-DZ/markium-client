# Markium — Content Schedule & Daily Checklist

A practical daily + weekly checklist for executing the social media warming plan. Print it, pin it, or copy it into Notion. The goal: know exactly what to do each day without thinking.

---

## The Rhythm

- **Sunday** = Production day (3-4 hours, heavy work)
- **Monday–Saturday** = Engagement + posting (30-45 min/day, light work)
- **Saturday evening** = Weekly review (1 hour)

This rhythm is sustainable. Don't try to produce content daily — you'll burn out.

---

## WEEKLY CHECKLIST — Every Sunday (Production Day)

Block 3-4 hours. Turn off notifications. This is your highest-leverage work of the week.

### Block 1: Plan the Week (30 min)
- [ ] Open last week's analytics — what worked, what didn't
- [ ] Feed data to Claude/ChatGPT for insights
- [ ] Review content calendar template
- [ ] Decide this week's theme (e.g. "Week of Profitability")
- [ ] Generate 30 content ideas with AI, pick the top 12-15

### Block 2: Scripts & Captions (1 hour)
- [ ] Write/AI-draft scripts for 5-7 video pieces
- [ ] Adapt all scripts to Darija (inject local phrases, voice)
- [ ] Write captions for each post (3 variations, pick best)
- [ ] Prepare hashtag sets (broad + niche + trending)

### Block 3: Video Production (1 hour)
- [ ] Record 1 long video (20-30 min tutorial in Darija)
- [ ] Record 3-5 short talking-head clips
- [ ] Run long video through Opus Clip → auto-extract 10-15 shorts
- [ ] Process all videos in CapCut: captions, branding, music

### Block 4: Visual Content (30 min)
- [ ] Create 2-3 carousels in Canva AI (educational posts)
- [ ] Create 5-7 static graphic posts
- [ ] Design thumbnails for YouTube videos
- [ ] Resize everything for each platform (Magic Resize)

### Block 5: Schedule Everything (30 min)
- [ ] Upload all content to Metricool / Meta Business Suite
- [ ] Schedule FB Page posts (Mon-Sun)
- [ ] Schedule Instagram posts + stories
- [ ] Schedule TikTok posts (may need manual posting)
- [ ] Upload YouTube long video + shorts
- [ ] Double-check all times (8-11 PM = best window)

### Block 6: Week Setup (15 min)
- [ ] Update content calendar — mark as "scheduled"
- [ ] Prep engagement targets: 5 FB groups to comment in
- [ ] Note any community questions to address this week
- [ ] Set phone reminders for daily engagement windows

**✅ Sunday done. You're ahead of the game.**

## DAILY CHECKLIST — Monday through Saturday

Each day takes ~30-45 minutes. Do it in one block or split morning/evening.

### Morning Routine (15 min) — Between 9-10 AM
- [ ] Check overnight notifications across all platforms
- [ ] Respond to any DMs (personal replies only — ManyChat handles FAQ)
- [ ] Check if scheduled posts went live (auto-fix any failures)
- [ ] Quick scan of Facebook e-commerce groups — note hot topics

### Midday Routine (10 min) — Between 12-1 PM
- [ ] Check morning post engagement
- [ ] Reply to every comment within 1 hour of post going live
- [ ] Quick-engage: like + comment on 5 posts from Algerian e-commerce creators

### Evening Routine (20 min) — Between 8-10 PM (peak window)
- [ ] Post manually if any platform requires it (TikTok often best manual)
- [ ] Respond to evening comments across all platforms
- [ ] Post daily story on FB + IG (behind-the-scenes, poll, or quick tip)
- [ ] Engage in 2 Facebook groups — leave 3-5 genuine comments each
- [ ] Reply to the Markium Facebook Group posts
- [ ] Check analytics: which post is winning today?

**✅ Daily done. Close the laptop.**

## WEEKLY REVIEW CHECKLIST — Saturday Evening (1 hour)

Do this **before** Sunday production. It informs what you create next.

### Data Collection (15 min)
- [ ] Export TikTok analytics (views, likes, shares, completion rate)
- [ ] Export Facebook Insights (reach, engagement, best post)
- [ ] Export Instagram Insights (reach, saves, shares)
- [ ] Export YouTube analytics (views, watch time, subs gained)
- [ ] Note growth: followers gained per platform

### Analysis (20 min)
- [ ] Paste all data into Claude/ChatGPT with prompt:
  ```
  Here's this week's social media performance: [data]
  1. Which posts performed best and why?
  2. What patterns do you see in timing, format, topic?
  3. What should I do more of next week?
  4. What should I stop doing?
  ```
- [ ] Identify top 3 winning posts across all platforms
- [ ] Identify 2 losers — note what didn't work
- [ ] Check which content pillar drove most engagement

### Planning (15 min)
- [ ] Decide next week's theme
- [ ] Note 5 content ideas from the analysis
- [ ] Flag any engagement wins to double down on
- [ ] Update KPI tracker (followers, engagement rate, DMs, signups)

### Admin (10 min)
- [ ] Back up all content files
- [ ] Clean up content calendar (archive completed items)
- [ ] Note any community wins to highlight next week

**✅ Week closed. Ready for Sunday production.**

## MONTHLY CHECKLIST — Last Sunday of Each Month

Deeper strategic review (add 1 hour on top of regular Sunday).

- [ ] Review 4 weeks of analytics — trend analysis
- [ ] Update KPI dashboard against targets
- [ ] Identify top 10 performing posts of the month
- [ ] Identify top 5 most engaged commenters (thank them, build relationships)
- [ ] Review competitors: what's working for them?
- [ ] Update content pillars based on what's resonating
- [ ] Adjust posting times based on audience activity
- [ ] Review AI prompt library — update with new winning prompts
- [ ] Plan next month's theme and major content pushes
- [ ] Review the Facebook Group — health check, posting frequency

## EMERGENCY / CATCH-UP CHECKLIST

If you miss a week or fall behind, don't try to catch up. Reset with this mini-version:

### Minimum Viable Week (when time is tight)
- [ ] 3 TikTok posts (not 5-7)
- [ ] 3 Facebook posts (not 6-7)
- [ ] 3 Instagram posts
- [ ] Daily engagement: 15 min total (not 30-45)
- [ ] Skip YouTube this week
- [ ] Keep the Facebook Group active with 1 post

**Rule**: Consistency > volume. 3 great posts/week beats 10 rushed ones.

## THE GOLDEN RULES

1. **Sunday is sacred.** Protect production day. No meetings, no distractions.
2. **30-45 min/day max** on weekdays. If it takes longer, you're doing it wrong.
3. **Post during peak window** (8-11 PM Algeria time) whenever possible.
4. **Reply to comments within 1 hour** of posting — biggest algorithm boost.
5. **Batch, don't improvise.** If it's not on the calendar, it doesn't happen.
6. **Track weekly, optimize monthly.** Don't make big strategy changes based on 1 post.
7. **Rest Sunday evening.** You produced, you earned it.
